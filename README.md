##### REST :

##### @GET

###### `/rest/city/{cityname}` - eg Szczecin, Moskwa, Krakow, California


###### `/rest/coordinate/{lat}/{lon}` - coordinates like e.g. {14,55/53,43} = Szczecin 


###### `/rest/zipcode/{zipNumber}/{countryCode}` eg - /70-366/pl
